React App - Robofriends
This is my first web app that I built using ReactJS. It features newer JavaScript standards (ES7 & Babel). The Robofriends app pulls user data from a JSON object and generates a randomized robot avatar for each person, which is then displayed in a card component. It also features a fully working search box at the top that filters which cards are displayed depending on the name that is searched.
